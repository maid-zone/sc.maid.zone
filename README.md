# sc.maid.zone
[soundcloak](https://github.com/maid-zone/soundcloak), but with extra patches for [sc.maid.zone](https://sc.maid.zone)

## Hosting
Hosted on [fly.io](https://fly.io), which means the traffic goes through their reverse proxies.

Configuration: 1 shared CPU, 256mb RAM, in Amsterdam, Netherlands

### Domains
- [sc.maid.zone](https://sc.maid.zone)
- [soundcloak.fly.dev](https://soundcloak.fly.dev)

<sub><i>Both are the same host, just different domains in case there are any issues</i></sub>

## Changes
TrustedProxyCheck is disabled.
EarlyData is enabled.
The page design is changed (accent color and more)

## What is stored
I store nothing. Only errors are temporarily logged (error description, the request parameters: query, username, track, etc)
